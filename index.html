My Project 1: ANRAZ Blood Bank

The Anraz Blood Bank Management System is a web-based application designed to enhance efficiency of blood bank operations, specifically for administrative management. Developed using PHP and MySQL, this system simplifies the process of managing blood bank functions, ensuring accurate data storage and quick access to vital information. Admins can manage inventory, patient records, test bookings, counter operations, and generate management reports. Forms for adding and updating records make it simple to input and modify data, while the report section helps in internal analysis and decision-making. The system prioritizes accuracy, smooth operation, and timely availability of data, making blood bank management efficient and reliable.

My Project 2: Museum Management System

This project represents the foundation of a museum management system through interrelated classes, implementing all four OOP pillars: Encapsulation, Polymorphism, Abstraction, and Inheritance. It covers employees, exhibits, artifacts, tickets, schedules, and visitors. The system has 13 classes, including an abstract class, with separate GUIs for Admin and Visitors. Admin can manage employees (tour guides, volunteers, security, janitors) and exhibits, while visitors can buy tickets, view available artifacts, and interact with the system. With MS Access database connectivity, visitor details are saved and accessible to Admin. The project aims to provide a modern, user-friendly platform to preserve and present cultural artifacts efficiently.

My Project 3: SOZE Art Gallery

SOZE Art Gallery is a responsive HTML and CSS-based website featuring a navbar, multiple linked pages, artists, events, reviews, and a gallery of artworks. Images on the homepage serve as navigation links, each leading to detailed reviews. CSS is used extensively for styling, transitions, and layout design to create an elegant and engaging user interface. The gallery offers 9 images on the homepage, with separate review pages for each art piece. The goal of this project is to provide art lovers with an interactive platform to explore and appreciate artwork in a visually appealing and user-friendly environment.
